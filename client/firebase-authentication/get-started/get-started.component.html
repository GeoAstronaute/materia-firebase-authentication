Prerequisites

This addon uses firebase-admin npm package under the hood.
To use the Firabase Authentication Addon for Materia Designer, you'll need a Firebase project, a configuration file with your service account's credentials and a firabase database URL.
Create Firebase project

If you don't already have a Firebase project, add one in the Firebase console . The Add project dialog also gives you the option to add Firebase to an existing Google Cloud Platform project.

Service account credential
  • - Navigate to the Service Accounts tab in your project's settings page.
  • - Click the Generate New Private Key button at the bottom of the Firebase Admin SDK section of the Service Accounts tab. After you click the button, a JSON file containing your service account's credentials will be downloaded. You'll need this to initialize the addon.
Database URL

You can find your database name on the Database page of your Firebase console project.
