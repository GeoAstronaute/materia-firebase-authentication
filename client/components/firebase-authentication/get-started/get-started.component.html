To use Firebase Authentication Addon for Materia you will need:
  • a Firebase account,
  • a Firebase project,
  • and a service account's credentials json file.

Prerequisites

- Get a Firebase account

- Create a Firebase project

- Get your Service account credential

  • Click the Generate New Private Key button at the bottom of the Firebase Admin SDK section, a JSON file containing your service account's credentials will be downloaded. You need this file to initialize the addon.
